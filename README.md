# book-seeker-android
Find books with Itunes API

# Development 

- Clean Architecture + MVVM Pattern
- Kotlin Language
- Main libs : RxAndroid, Retrofit, Navigation
- Koin Dependency injection 
- Offline support using cache
- Modularity [ domain, data, commons, app ]
